# Café Website

A simple café website built using **HTML** and **CSS**.  
Currently **desktop layout only**; responsive design coming soon.

---

## 🏠 Features

- Sections for:
  - Home / Hero
  - Menu / Products
  - Special Deals
  - Delivery information
  - Fresh, organic products  
- Modern, clean design  
- Built with **HTML and CSS only**  
- Easy to customize for other cafés or small businesses  

---

## 💻 Technologies Used

- **HTML5** – Semantic markup for clean structure  
- **CSS3** – Styling for layout and design  
- Optional: **Font Awesome** for icons  

---

## 🛠 Installation

1. Clone the repository:

```bash
git clone https://github.com/AyaAhmed50/Cafe-shop
````

2. Open `index.html` in your browser:

```bash
open index.html
```

3. Customize content and styles in the `css/` folder

---

## 🌐 Folder Structure

```text
cafe-website/
│
├─ index.html
├─ about.html          # optional pages
├─ css/
│   └─ style.css
├─ images/
│   └─ logo.png, menu items, etc.
└─ README.md
```

---

## 🎨 Customization

* Change colors in `css/style.css` to match your café brand
* Update images in the `images/` folder
* Add or remove menu items easily in HTML
